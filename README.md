### nest-admin

**基于NestJs + TypeScript + TypeORM + Redis + MySql + Vue3 + Ant Design Vue编写的一款简单高效的前后端分离的权限管理系统。希望这个项目在全栈的路上能够帮助到你。**

- 前端项目地址：[传送门](https://github.com/shandongtlb/vue3-admin-front)

### 项目启动前的准备工作
- sql文件：[/sql/init.sql](https://github.com/shandongtlb/vue3-admin-front/tree/main/sql)
- mysql和redis连接配置：[src/config/config.development.ts](https://github.com/shandongtlb/vue3-admin-front/blob/main/src/config/config.development.ts)
